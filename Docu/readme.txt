
Homemade Geiger Counter and Dosimeter


How it works: 

http://www.youtube.com/watch?v=5IWn8H0pbQ4


For this project you need an AVR programmer to write the "GeigerCounter_v1.0.hex" file into the microcontroller.
(Ex. AVR programmer project in avr_isp_eng.zip)

First program the Fuse Bits and then the hex file. (Fuse bits settings attached).
